About the website

About Us

This is a web-page that started in 2019 with the intentions of simplifying learning and increasing the learning outcome in the course TPK5100 at Norwegian University of Science and Technology. We think that digitizing will benefit both the lecturer and students.

With "The Road to Knowledge" it is simple to learn the basic terminology and theory through a low-threshold multiple-choice quiz. The purpose for the website is to act as a proof of concept. As this website is quite basic, it serves the purpose of proving that the creation of a website for the course TPK5100 is managable.

This page is free to use for everyone. The book used as a curriculum is "Veien til suksess" by Bassam Hussein, 2016. The main reason for initiating the web-page was a project delivery in the course TPK5100. It has been made by 4 students with no prerequisite competance in website creation or management. The website is made in HTML, while the quiz is made using javascript.
